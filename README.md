# 📊 E-commerce Sales Analytics Project

## 🔹 Overview
This project analyzes e-commerce order data to derive key business insights such as revenue trends, top categories, and top-performing states.

---

## 🔹 Tools Used
- SQL
- Python (Pandas)
- Power BI

---

## 🔹 Dataset
The dataset contains order-level data including:
- Order ID
- Date
- Category
- State
- Amount (Revenue)

---

## 🔹 Key Analysis Performed
- Total revenue calculation
- Top-performing product categories
- Top-performing states
- Data cleaning and preprocessing

---

## 🔹 Key Insights
- Certain categories contribute majority of revenue
- A few states dominate overall sales
- Revenue is concentrated in specific segments

---

## 🔹 Dashboard
Power BI dashboard is included in this repository.

---

## 🔹 Conclusion
This project helps understand sales performance and identify high-performing areas for business growth.
